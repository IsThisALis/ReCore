# RExample

Includes examples of ReCore usage
Only for study

## Buid & Run

### Clone repository 

```
git clone https://github.com/IsThisALis/ReCore.git
cd ReCore/recore-modules/rexample
```

### Build

```
mvn package -Dnative.target=natives-linux
```

### Run

```
java -jar target/rexample-0.0.4-alpha.jar
```
